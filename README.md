# Next.js Documentation

This directory contains the complete Next.js documentation downloaded from the official website.

## Structure

- **getting-started/** - Getting started guides and tutorials
- **guides/** - Comprehensive guides for various Next.js features
- **api-reference/** - API reference documentation
- **architecture/** - Architecture and design documentation
- **community/** - Community resources and contribution guidelines

## Usage

Navigate through the folders to find specific documentation topics. All files are in markdown format and can be viewed with any markdown viewer or editor.
